## Install

Clone the project :

```bash
git clone https://github.com/ThomasVogeleisen/P14-react.git
```

Install :

```bash
npm install
```

Launch :

```bash
npm run dev
```

## Testing

Run tests with Jest :

```bash
npm run test
```

## Performances

Before test performance with LightHouse, build the project :

```bash
npm run build
```
